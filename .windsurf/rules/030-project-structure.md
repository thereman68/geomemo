# Project Structure

- **src/**: Main source directory
  - **assets/**: Contains styling and images
  - **components/**: Reusable components
  - **json/**: Data files and configuration
  - **router/**: Routing configuration
  - **stores/**: State management
  - **use/**: Composable functions and utilities
  - **views/**: Main page components
- **public/**: Static assets.  Include large/binary files that don't need transforming or bundling and any files that need to maintain their original file names
